Portfolio

Hangman Game

Hangman is a guessing game to form a word. Player tries to guess a word by suggesting letters within a certain number of guesses.

For this web application, I have used Python flask Framework, render_template for jinja2 and request for form details. As this my first web application project I chose Flask as it is a lightweight framework.

Quiz App

This is a quiz application. Player can choose a category and each category has 5 questions. Player can either play, add or delete a quiz.

Main goal of this project is to create, update or delete some endpoints using Python's FastAPI Framework,jinja2 and HTML template. Quizzes are stored in a database using sqlite3.


Webscraping Project

This application is scraping the Linkedin website's jobs for keyword "Python developer and location "Munich". It extracts the HTML tags for the information of job title, company name, job link and published date. Finally, a gragh is drawn for the number of job entries for 3 months.

BeautifulSoup library is used for pulling data out of HTML and pandas module for data manipulation and analysis. Visualisation is done with matplotlib.

World Happiness Report

This is a detailed data analysis on the World Happiness Report for the years 2015-2020. Emphasize the relation between the happiness score and the geographical location.

NumPy and pandas are used for data manipulation and analysis. Graphs and charts are plotted by using matplotlib and seaborn. Dashboards were created using Tableau software.
